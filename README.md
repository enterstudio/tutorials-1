on a number of different email clients:

- Evolution
- Geary
- Gmail (Mobile)
- K-9 Mail
- KMail
- Windows 10 Mail
- Mailbird
- Mailpile
- MUTT
- Sylpheed
- The Bat
- Thunderbird
